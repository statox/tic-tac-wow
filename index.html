This is my Tic Tac Toe based games playground

Tic Tac Cup is a tic tac toe game with pieces of different dimensions, inspired by Gobblet In the "Game" section you can play against different level of AIs. The "Runner" section allows to make the AIs play games against each other and get statistics of these games.

The Tic Tac Toe page is unsurprisingly a classical tic tac toe game with a random AI.

The Tic Tac Toe 2 page is more interesting as this is a complete tic tac toe implementation with different set of AIs and a "Runner" section to benchmark the AIs against each other. a random AI.
